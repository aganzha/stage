Stage

Stage is GTK4 git client inspired by Magit. Pretends to be Magit without Emacs.

Installation

Flatpak - is the proper way to install Stage on your linux desktop. Further commands are for terminal.

Add Flathub to flatpak remotes: 

      flatpak remote-add --if-not-exists flathub https://dl.flathub.org/repo/flathub.flatpakrepo
Install Stage 

      flatpak install flathub io.github.aganzha.Stage
Run Stage 

      flatpak run flathub io.github.aganzha.Stage
Update 

      flatpak update io.github.aganzha.Stage

Usage

Main screen - Status view

Open repository either by clicking placeholder button, or repository list button in the header bar. Stage, then, will live track all changes you do on files inside repository and display in Status view in form of text diff.

Highlighted line in Status view behave like a cursor in TUI apps. Move cursor with arrows or by mouse clicking in any area. Commands your issued for Stage are applied to "thing" under cursor. E.g. to stage (git add) file put cursor on file name and press s, or click in header bar. Whole file will be added to staging are for further commit.

Main commands are:

  • s - Stage. Also BENTER could be used. Button in headerbar -
  • u - Unstage. Button -
  • k - Kill (discard changes)
  • TAB - Expand/Collapse. Also SPACE could be used.
Commands does not related to cursor position:
  • c - Commit. Button in headerbar -
  • p - Push. Button -
  • f - Pull (as in Fetch). Button -
Command for displaying additional windows:
  • b - Branches view
  • l - Log view
  • z - Stash panel
  • t - Tags view
